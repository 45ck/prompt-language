# Use Cases (Bounded CRM MVP)

## UC-001 — Authenticate and access the app

- User can log in and access CRM features.
- User can log out and loses access until logging in again.

## UC-010 — Manage contacts

- Create a contact with required name and optional details (email/phone).
- Search and view a contact.
- Update a contact.

## UC-020 — Manage companies

- Create a company with required name.
- Search and view a company.
- Update a company.

## UC-030 — Manage opportunities and pipeline

- Create an opportunity linked to a company (and optionally a primary contact).
- Assign stage and owner.
- Move opportunity between stages.
- View opportunities by stage (board or list).

## UC-040 — Manage tasks

- Create tasks linked to a contact, company, or opportunity.
- Set due date (optional), mark complete/incomplete.
- View overdue tasks and due-soon tasks.

## UC-050 — Manage notes

- Add a note linked to exactly one parent record (contact/company/opportunity).
- View notes for a record in reverse chronological order.

## UC-060 — Dashboard

- See overdue tasks and tasks due soon.
- See opportunity counts by stage.

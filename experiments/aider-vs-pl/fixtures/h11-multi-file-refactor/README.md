# Contact Manager

A simple Contact management system for storing and retrieving Contact records.

## Modules

- `contact.js` — Contact class with name, email, phone, company fields
- `contact-store.js` — In-memory ContactStore for CRUD operations
- `routes.js` — Route handlers for Contact API endpoints
- `seed.js` — Seed data with sample Contact records
- `test.js` — Unit tests for Contact and ContactStore

## Usage

```bash
node src/app.js    # Run the application
node src/test.js   # Run tests
```

# Solo Prompt — Marketing Website

Build a complete marketing website for **CloudPulse**, a cloud monitoring SaaS product.

## Product Info

- **Product Name**: CloudPulse
- **Tagline**: "Monitor smarter, respond faster."
- **Brand Colors**: Primary #2563EB (blue), Secondary #7C3AED (purple), Accent #10B981 (green), Dark #1E293B, Light #F8FAFC

## Requirements

Create a single `index.html` file with all HTML and CSS inline (no external files). The website must include:

1. **Navigation**: Sticky nav bar with links to all sections
2. **Hero Section**: Headline, subheadline with tagline, CTA button
3. **Features Section**: At least 3 feature cards with descriptions
4. **Pricing Section**: At least 2 pricing tiers with feature lists and CTA buttons
5. **Testimonials Section**: At least 2 customer quotes with name and company
6. **Footer**: Contact info and social media links (Twitter, LinkedIn, GitHub)

## Technical Requirements

- Valid HTML5 with DOCTYPE
- Responsive meta viewport tag
- CSS media queries for mobile (max-width: 768px)
- All navigation links use anchor IDs
- No external images (use CSS or inline SVG)
- Professional tone, no exclamation marks in body copy
- Title Case headings
- Action-oriented CTA buttons (e.g., "Get Started", "Start Monitoring")

Write the complete index.html file now.

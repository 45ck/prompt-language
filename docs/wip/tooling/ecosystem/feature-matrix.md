# Feature Matrix

| Project                  | Layer                         | Core primitives                                                                   | Strong ideas                                     | Best fit for prompt-language                      |
| ------------------------ | ----------------------------- | --------------------------------------------------------------------------------- | ------------------------------------------------ | ------------------------------------------------- |
| prompt-language          | supervision runtime           | gates, variables, loops, review, ask, spawn/await, memory, imports                | runtime owns supervision rather than reasoning   | baseline                                          |
| Archon                   | orchestration shell           | YAML DAG workflows, commands, artifacts, fresh context, approvals, worktrees      | fresh session boundaries and artifact chains     | import fresh-context and artifact bootstrap ideas |
| Druids                   | distributed event runtime     | async agent programs, events, sandboxed VMs, judge/worker, builder/critic/auditor | verifier hierarchies and best-of-N patterns      | import verifier patterns and examples             |
| Hankweave Runtime        | headless long-run runtime     | codons, rigs, sentinels, budgets, preflight, checkpoints, event journal           | debuggability, repairability, context boundaries | import preflight and event journal thinking       |
| GitHub Agentic Workflows | compiled workflow shell       | Markdown plus frontmatter compiled to Actions YAML, safe outputs, permissions     | explicit permissions and compile boundary        | prototype compile-target and safe-output ideas    |
| Streetrace               | multi-agent runtime plus DSL  | DSL agent definitions, tools, retries, timeouts, constraint handlers              | policy hooks as architecture                     | import policy-hook ideas carefully                |
| Agent Loom               | durable artifact protocol     | rules, skills, commands, tickets, packets, memory                                 | packetized handoff and visible truth layers      | import packet and handoff patterns                |
| Doctrine                 | source language plus compiler | doctrine source, inheritance, typed inputs and outputs, compile to docs           | source/runtime split and fail-loud compilation   | reference for a future higher-level source layer  |
| Context Cascade          | selective-load hierarchy      | playbooks, skills, agents, commands, routing, session manager                     | load only what you need                          | prototype selective loading carefully             |
| IBM PDL                  | declarative prompt language   | YAML programs, types, loops, functions, structured decoding                       | strong type and shape orientation                | reference for typed output ideas                  |
| AgentScope Runtime       | deployment substrate          | sandboxes, APIs, deploy modes, observability, session state                       | serving and deployment substrate                 | reference only, mostly out of core scope          |

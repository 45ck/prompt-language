# Success metrics (Discovery targets)

Metrics focus on whether the bounded CRM MVP creates daily workflow value for SME teams.

## Activation (time-to-first-value)
- **TTFV:** time from first login to first created contact/company (goal: minutes, not hours).
- **First workflow completion:** contact/company + note + task created in one session.

## Engagement (weekly usage)
- **Weekly active users (WAU):** users who create or update at least one record per week.
- **Core actions per active user:** notes created, tasks completed, opportunities advanced.

## Data quality (pipeline hygiene)
- **% opportunities with a next-step task** (open task linked to the opportunity).
- **% opportunities with stage set** (target: ~100%).
- **Stale opportunities:** count of opportunities not updated in N days (threshold chosen by team).

## Productivity (follow-through)
- **On-time task completion rate:** tasks completed by due date.
- **Overdue backlog trend:** weekly change in overdue tasks.

## Visibility (dashboard usefulness)
- **Dashboard visit frequency:** percent of users opening dashboard on workdays.
- **Decision support:** qualitative feedback: “Can you answer what’s stuck and what’s next?”

## Qualitative success signals
- “We stopped using spreadsheets for pipeline.”
- “Handoffs work without meetings because notes/tasks are in one place.”
- “We can review the week’s pipeline in 10 minutes.”

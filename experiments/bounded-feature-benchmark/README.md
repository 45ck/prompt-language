# Bounded Feature Benchmark

Use this folder for experiments that measure whether a bounded implementation can complete a target feature faster or more reliably than a broader alternative.

This benchmark is part of the non-factory proof program:

- compare bounded PL supervision against a simpler baseline on medium-complexity
  feature work
- score not just "did code appear," but feature correctness, tests, migration
  safety, auth correctness, and regressions

# Language Server (WIP)

> **WIP: not implemented yet.** This page describes intended behavior, not a released server.

## Goal

Provide editor-agnostic prompt-language tooling through the Language Server Protocol.

## Intended features

- hover docs for keywords
- autocomplete for nodes and built-in variables
- go-to-definition for variables
- inline diagnostics from the linter
- code actions for common fixes

## Current workaround

Use the docs plus CLI linting, or rely on a future VS Code-only experience.

Features

Everything You Need to Monitor Your Infrastructure

A complete observability platform purpose-built for modern cloud teams. Real-time metrics, intelligent alerting, and powerful dashboards — all without the operational complexity.

Real-Time Monitoring

Continuous streaming telemetry from every connected cloud service, with sub-second latency data pipelines delivering live CPU, memory, network throughput, and disk I/O metrics directly to your dashboards. Never miss a spike, dip, or anomaly across your entire fleet. Supports StatsD, OpenTelemetry, Prometheus scraping, and our native lightweight agent — all in one place.

Intelligent Alerting

Machine-learning anomaly detection combined with fully configurable threshold-based rules. Route alerts to Slack, PagerDuty, Opsgenie, email, or any webhook endpoint. Define escalation policies, on-call rotations, and suppression windows to eliminate noise and alert fatigue. The right person is notified with the right context — every time, without exception.

Custom Dashboards

Drag-and-drop dashboard builder with over 50 widget types: time-series charts, heat maps, topology graphs, SLO burn-rate indicators, log panels, and more. Share dashboards across teams, set granular view permissions, or embed them in your internal portals via the embed API. Version-control dashboards as JSON and deploy them through Terraform and CI/CD pipelines.

Multi-Cloud Support

Native integrations with Amazon Web Services, Microsoft Azure, and Google Cloud Platform. CloudPulse normalizes metrics from disparate providers into a single unified data model, enabling true cross-cloud performance comparison, consolidated cost visibility, and side-by-side analysis. Hybrid and on-premise environments are also fully supported via the agent and OpenTelemetry collector.

Team Collaboration

Role-based access control, shared annotation timelines, and a built-in incident war-room experience for coordinating real-time response across your team. Invite unlimited read-only viewers on all paid plans. Manage permissions at the organization, project, or individual dashboard level. Integrate with Jira and GitHub Issues for automatic ticket creation when critical alerts fire.

API Access

A fully documented REST and GraphQL API with official SDKs for Python, Node.js, Go, and Ruby. Automate dashboard provisioning, manage alert rules programmatically, and pipe metrics into your own data warehouse or BI tooling. Our Terraform provider lets you treat your entire monitoring configuration as code — checked into version control and deployed through any standard CI/CD pipeline.

FAQ

Frequently Asked Questions

Everything you need to know before getting started. Can't find your answer? Reach out to our team — we respond within one business hour on weekdays.

How does the 14-day free trial work?
When you sign up, your workspace is automatically placed on the Pro plan for 14 days with no credit card required. You have full access to every Pro feature — including unlimited alert rules, custom dashboards, Slack and PagerDuty notifications, and ML anomaly detection. At the end of the trial period you can choose any paid plan or continue on the free community tier. Your dashboards, alert rules, and historical data are preserved regardless of which plan you select.
Which cloud providers does CloudPulse support?
CloudPulse natively integrates with Amazon Web Services, Microsoft Azure, and Google Cloud Platform. We support over 120 individual services across these three providers, including compute, storage, managed databases, serverless functions, and container orchestration platforms. Hybrid and on-premise environments are supported via our OpenTelemetry-compatible agent. New service integrations are shipped on a monthly release cadence, and you can request integrations directly from the product roadmap portal.
How is pricing calculated for large environments?
Pricing is based on the number of monitored hosts — virtual machines, container nodes, or bare-metal servers — within your workspace. Serverless functions and managed services such as S3, Blob Storage, and Cloud SQL do not count toward your host limit. Enterprise customers receive custom volume pricing negotiated directly with our sales team, typically with significant discounts for larger fleet sizes. All plans include unlimited metrics ingestion resolution and unlimited team members at no additional charge.
Is CloudPulse SOC 2 Type II certified?
Yes. CloudPulse has maintained SOC 2 Type II certification since 2024. Our infrastructure is hosted exclusively in ISO 27001-certified data centers. All data in transit is encrypted with TLS 1.3, and all data at rest is encrypted with AES-256. We are also GDPR compliant and offer EU data residency for European customers. Enterprise customers may request a copy of our latest audit report and penetration test summary under NDA by contacting their dedicated account manager.
Can I export my data if I decide to leave?
Absolutely. You own your data, full stop. CloudPulse provides a complete data export API and a self-service export tool in workspace settings that allows you to download all metrics, alert history, dashboard configurations, and annotation data in JSON or CSV format at any time — including during a free trial. We will never hold your data hostage or impose exit fees. After account closure your data is retained for 90 days before permanent deletion, giving you ample time to complete any data migrations.
What monitoring agent do I need to install, and how resource-intensive is it?
The CloudPulse Agent is a single statically-compiled binary, available for Linux (amd64, arm64), Windows Server, and macOS. It consumes less than 50 MB of RAM and under 1% CPU at rest on standard instances. The agent supports autodiscovery of running processes, containers, and system metrics with zero configuration required out of the box. You can also use our fully agentless mode for cloud-provider metrics by connecting via read-only API credentials — meaning no software installation is needed at all for cloud-native environments.

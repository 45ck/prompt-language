# Self-Healing CI

Use this folder for experiments that test whether the system can repair failing checks or infrastructure issues with minimal human intervention.

This benchmark is part of the non-factory proof program:

- prove a narrow workflow where prompt-language supervision should be naturally
  strong
- keep the task bounded around real CI gates
- measure repair success, false fixes, regression introduction, and time to
  green

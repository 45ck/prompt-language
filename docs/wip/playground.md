# Web Playground (WIP)

> **WIP: not implemented yet.** This page describes intended behavior, not a released site.

## Goal

Let new users try prompt-language in the browser without installing Claude Code or the runtime.

## Intended features

- Monaco editor for flow text
- parsed AST view
- rendered flow preview
- inline lint feedback
- dry-run execution without real AI calls
- shareable URLs for example flows

## Current workaround

The fastest path today is still local install plus `demo`, `init`, and the docs examples.

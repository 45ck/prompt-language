# Parallel Planning

Use this folder for experiments that compare single-threaded planning with explicit parallel decomposition and coordination.

This benchmark is part of the non-factory proof program:

- test whether specialist fan-out helps before implementation starts
- compare single-agent planning against bounded `spawn` / `await` planning
- measure completeness, missed acceptance criteria, implementation readiness,
  and coordination overhead

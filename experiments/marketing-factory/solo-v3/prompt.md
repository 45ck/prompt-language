Build a marketing website for CloudPulse, a cloud monitoring SaaS platform.

Product: CloudPulse ("Monitor smarter, respond faster.")
Brand colors: #2563EB (primary), #7C3AED (secondary), #10B981 (accent), #1E293B (dark), #F8FAFC (light)

Steps:

1. Create brand-guidelines.md covering voice/tone rules, the color palette, typography, and naming conventions.
2. Create content-brief.md planning the sections, copy, and structure for the site. Decide on features, pricing tiers, testimonials, and CTAs. All content must follow the brand guidelines.
3. Build the website as a single self-contained index.html. Read brand-guidelines.md and content-brief.md for all decisions. Requirements: semantic HTML, responsive CSS with media queries, CSS custom properties for colors, full SEO (meta tags, Open Graph, JSON-LD, canonical, favicon), accessibility (lang, ARIA, heading hierarchy), no external dependencies, no <img> tags.

# Guides

Tutorials and conceptual walkthroughs for learning and using prompt-language.

Guides are the product-first onboarding path for the shipped runtime. They teach prompt-language through step-by-step explanations and opinionated paths. For exact syntax, limits, defaults, and edge cases, use the [Language Reference](../reference/index.md). If a topic appears only in the [Roadmap](../roadmap.md) or [WIP Features](../wip/index.md), treat it as not available today.

# Results

Store experiment run outputs, score summaries, and analysis artifacts here.

Use one subfolder or file set per experiment so results stay easy to compare and archive.

For the productized eval runner, this is the canonical home for locked baselines and saved comparison reports. Ad hoc script history under `scripts/eval/results/` is useful evidence, but it is not the canonical baseline bank.

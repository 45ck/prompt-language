# Solo prompt (no PL structure)

Build a complete marketing website for NightOwl, a sleep tracking platform for knowledge workers.

## Product brief

NightOwl helps knowledge workers (engineers, researchers, writers) understand how sleep affects their cognitive performance. Core features: passive sleep tracking, cognitive performance correlation, team insights for managers, AI recommendations based on chronotype, and integrations with GitHub/Linear/Notion/Calendar/Slack.

Brand: intelligent but approachable, data-driven not clinical, modern minimal, dark-mode-first, trustworthy with health data. Tagline: "Sleep smarter, think sharper."

Target: tech-savvy professionals aged 25-45, remote workers with irregular schedules.

## Requirements

1. Build a complete, production-quality marketing website
2. Include: hero section, features overview, how-it-works, social proof/testimonials, pricing tiers, integrations showcase, FAQ, footer with links
3. Must be responsive, accessible (WCAG 2.1 AA), and performant
4. Must pass `npm run build` and `npx eslint . --max-warnings 0`
5. Use a real framework and project structure (not a single HTML file)
6. Dark-mode-first design fitting for a sleep product
7. Premium, trustworthy feel

Build everything from scratch — choose the tech stack, design system, content, and structure yourself. Make it production-ready.
